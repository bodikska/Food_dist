# Food Delivery Interface

Frontend interface for a food delivery service built with vanilla JavaScript.

The application displays food items on the page and demonstrates a simple interactive interface for browsing products.

## Tech Stack

JavaScript (Vanilla JS)  
HTML  
CSS  

## Features

- display food items as cards
- dynamic rendering of products
- simple user interface
- styled layout using CSS
- interactive elements with JavaScript

## Functionality

The application renders food items on the page and displays them as product cards.

JavaScript is used to dynamically manipulate the DOM and manage interface interactions.

Users can browse the available food items through a simple and clean interface.
